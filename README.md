# 写在开头

##关于写作目的

这是一个学习Objective-C的教程，写作的原因很简单，仅仅是因为我想用用gitbooks而已。

内容包含OC语言部分，关于iOS开发过程中的技巧与注意点，我将它记录在了我的博客中[BRIGHTTJ](http://www.brighttj.com)。而与本教程同步的视频教程，也正在录制当中，到时会将连接附上。
